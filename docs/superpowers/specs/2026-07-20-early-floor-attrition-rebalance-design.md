# 序盤フロアの消耗バランス再設計

日付: 2026-07-20
対象: 第一エリア「崩れた坑道」(B1〜B5) / 基本4職 (戦士・盗賊・僧侶・魔術師)

## 問題

序盤の消耗が激しく、第一の迷宮を突破できない。

## 診断

`scratch/sim_solo_b1f.js` による実測 (N=800、歩数レンジ40〜70、MILWAなし、ボス戦なし)。

| 職業 | 突破率 | 平均勝利戦闘数 | MP枯渇経験率 |
| --- | --- | --- | --- |
| 戦士 | 60.4% | 2.82 | 0.0% |
| 盗賊 | 34.5% | 2.02 | 0.0% |
| 僧侶 | 41.6% | 2.19 | 32.0% |
| 魔術師 | 37.3% | 2.13 | 1.0% |

読み取れること:

1. **3戦目で死ぬ**。B1F単体で約40〜65%が死亡。
2. **状態異常も敵魔法も主因ではない**。第一エリアのプールは既に薄い
   (まどろみ胞子 `statusChance` 0.2 / フラッシュバット盲目 / 泥の呪い子
   DEF-2、攻撃魔法はゴブリンの呪術師 HALITO 0.3 のみ)。
3. **職間格差が26pt**。原因は近接倍率 (`getMeleeModifiers`)。職の物理弱さは
   STR差と装備制限 (`classes[]`) で既に二重に課されており、倍率は三重目。
4. **魔術師には倍率が効いていない**。杖atk1・STR7でダメージ式が下限1に張り付く。
   MP枯渇率1.0%が示すとおり、MPを使い切る前に死んでいる。

## 変更内容

### 1. 近接倍率の廃止

`src/combat_logic/damage.js:15` `getMeleeModifiers` を全職 1.00 にする。

職の物理差は STR と装備制限で表現する。倍率は同じ差の三重掛けなので削る。

### 2. HALITO のダメージ引き上げ

`src/systems/spell_effects.js:78` `Math.floor(rng() * 11) + 8` →
`Math.floor(rng() * 11) + 12` (8-18 → 12-22)。

第一エリアの敵HP12〜22を1発圏内に入れ、魔術師の被弾ターン数を半減させる。

### 3. 杖持ちの通常攻撃を魔力弾化

`src/combat_logic/round.js:244` 付近。Mage / Bishop の物理攻撃時、通常の物理
ダメージと INT 基準のダメージの大きい方を採用する。

```
bolt = max(1, floor(INT / 3) + rand(0..2) - floor(敵DEF / 4))
dmg  = max(dmg, bolt)
```

Lv1魔術師 (INT16) で 5〜7。MP切れ後に無力化する詰みを解消する。

### 4. 浅層の遭遇サイズを1体寄せ

`src/data/encounters.js` `ENCOUNTER_SIZE_WEIGHTS`:

- キー `1`: `[0.55, 0.42, 0.03]` → `[0.70, 0.30, 0.00]`
- キー `2`: `[0.40, 0.55, 0.05]` → `[0.55, 0.45, 0.00]`
- キー `3` 以降は変更しない。3体遭遇は深層の脅威として温存する。

完全な1対1化はしない。2体戦の「どちらから倒すか」はソロ戦闘の技術軸であり、
コアループ文書の pillar に含まれる。

### 5. 僧侶の初期MP引き上げ

`src/state/initial_state.js` の Priest プリセット: `mp` / `maxMp` を 5 → 7。

僧侶だけMP枯渇率が突出 (32%)。DIOSを撃てない僧侶は殴りの遅い戦士でしかない。

## 期待結果 (実測済み)

同条件 (N=800、40〜70歩、MILWAなし) で全変更を適用したときの計測値。

| 職業 | 変更前 | 変更後 |
| --- | --- | --- |
| 戦士 | 60.4% | 69.9% |
| 盗賊 | 34.5% | 71.6% |
| 僧侶 | 41.6% | 69.4% |
| 魔術師 | 37.3% | 67.1% |

- 基本4職の格差: 26pt → 4.5pt
- 全員が目標レンジ 65〜75% に収まる
- 僧侶のMP枯渇率: 32.0% → 17.0%
- 魔術師のMP枯渇率: 1.0% → 0.0%、平均勝利戦闘数 2.13 → 3.19

## 採用しなかった案

- **状態異常の削減**: 第一エリアは既に薄く、削っても効果が小さい。坑道の個性
  だけが失われる。
- **敵ATK/HPの一律引き下げ**: 全職に一律に効くため格差が残る。深層スケーリング
  にも波及する。
- **完全な1対1化**: kill order というソロ戦闘の技術軸を消す。確率調整で足りた。
- **魔術師のMPプール拡大**: MP枯渇率1.0%が示すとおり枯渇していない。効果が薄い。

## 範囲外

司教 (変更後も10.5%) と忍者 (36.8%) は上位職のため今回の対象外。特に司教は
近接0.50・杖atk1・MP3で、上記変更後も破綻したまま残る。別Issueで扱う。
