# 設計: ソロ深度ローグライトへの完全転換

日付: 2026-07-18
状態: 承認済み（設計のみ。実装は別途指示）

## 決定サマリ

現行本編（6人パーティ・固定マップ・町経済・B5F攻略）を廃止し、
**1人キャラで「どこまで深く潜れるか」を遊ぶ深度スコア型ローグライト**に
完全置き換えする。

確定した基本方針:

| 論点 | 決定 |
| --- | --- |
| 本編との関係 | 完全置き換え（別モード併設しない） |
| ラン時間 | 固定目標なし。中断セーブ前提で複数セッションに跨げる |
| メタ成長 | ハイブリッド: 毎ランLv1開始＋アンロック＋上限付き恒久ステ |
| 撤退と死亡 | 撤退＝素材満額 / 死亡＝3割 |
| 深度構造 | 無限スケーリング＋5フロア毎の節目 |
| 経済転生 | 案A: 鑑定・素材・工房をラン内/メタシステムへ転生 |
| 旧セーブ | 破棄。移行・補償なし |

既存設計docへの影響:

- `.agents/game-design-core-loop.md`: 全面改訂。
  **2026-07-10の「固定マップ・ラン毎再生成しない」決定を正式に廃棄**。
- `.agents/game-design.md`（町経済）: 廃棄し、本docのメタ経済へ置換。
- `.agents/game-design-equipment-builds.md`: アフィックス設計は
  ラン内ビルドの主役として存続。

## コアループ

```text
町（メタ画面）→ クラス・初期装備を選んで潜行（毎回Lv1）→
生成フロアを探索・戦闘・未鑑定装備で即席ビルド →
5フロア毎の節目（ボス・帰還ポータル・商人）→
「撤退して満額確保」か「もう1層」→
死亡なら3割持ち帰り → 素材で工房アンロック → 再潜行、前回より深く
```

深度＝スコアかつ進行度。自己最深記録をタイトル・町・リザルトに常時表示。

## ラン構造

- 開始: アンロック済みクラスから1つ選択、初期装備候補から数点選択。
  即潜行できる導線にする。
- フロア: 生成マップ。階段発見で次フロアへ。**フロア間は不可逆**
  （引き返し不可）。「潜る」一方向性を明確化する。
- 撤退手段:
  1. 節目フロア（5の倍数）の帰還ポータル常設。
  2. 「帰還の翼」的なラン内アイテム（有限。拾得・節目商人で入手）で
     どこでも撤退。
- 撤退＝素材満額＋記録確定。死亡＝その場でラン終了、素材3割、
  死亡深度も記録に残る。
- 中断セーブ: いつでも中断→完全再開。既存の遠征resume基盤
  （`src/state/save_payload.js` の `resolvePersistedGameState` 系）を転用。
  結果を見てからロードする悪用はオートセーブ上書きで防止。

## 深度と生成

- 無限スケーリング: 敵ステ・報酬はフロア数の関数
  （線形＋節目毎に係数段上げ）。
- バイオーム: 5フロア毎に切替（敵プール・ギミック・見た目。
  既存 `src/styles/floor-themes.css` を転用）。
  v1は手作りバイオーム6種≒30F分。以降は既存バイオームの
  高倍率再循環で割り切る。
- 節目フロア: ボス戦→帰還ポータル＋商人。「一息つく・記録を刻む・
  撤退判断する」場。
- 生成: 既存 `src/map_generator.js`＋`src/seed_rng.js` を
  ラン毎リシード。フロアテンプレ（サイズ・部屋数・ギミック密度）を
  深度で切替。
- **生成品質がv1最重要リスク**。実装順の先頭でプレイアブル検証する。

## 開始地点ショートカット

- 節目ボス撃破で「その節目から開始」をアンロック。
- 節目開始ランは素材収入に減額係数を掛ける。
  記録更新目的（節目開始）と素材稼ぎ目的（B1開始）で
  スタート地点を使い分ける判断を作る。

## ソロ戦闘（ターン制メニュー維持・全面再バランス）

- 行動経済: 敵グループは1〜3体に縮小。現行の6人前提編成
  （3〜5体）は廃棄。複数体戦は全体スペル・範囲アフィックス・
  処理順の判断で捌く。
- 状態異常再設計: ソロで被弾＝即詰みを禁止する。
  - 麻痺/睡眠: 最大1ターン＋被弾で解除。
  - 即死: 廃止。深層ボス限定の「大ダメージ＋抵抗ビルドで軽減」に置換。
  - 毒/呪い: 継続コスト型として維持。
- 敵ロール3種で編成を組む:
  - aggressor（殴る）
  - disruptor（状態異常・妨害）
  - amplifier（他の敵を強化。先に倒すか判断させる）
  ソロ戦の縦軸は「処理順」。
- 逃走: 確率制廃止→確定成功＋代償（1マス後退＋敵の攻撃1発）。
  逃げ判断を乱数で殺さない。
- 回復: 回復役不在の代替として、消耗品＋フロア移動時小回復＋
  回復系アフィックス。戦闘中回復は割高にしてヒール一辺倒を防ぐ。

## ビルドと鑑定ギャンブル（柱3の転生・本作の独自フック）

- 装備スロット構成は現行を維持。アフィックス（cores/supports）を
  ラン内ビルドの主役に昇格。
- **装備はラン内限り**。撤退・死亡いずれでもラン終了時に消滅し、
  持ち帰るのは素材のみ（装備持ち帰りは案Cとして棄却。v2再検討可）。
- 拾得装備は原則未鑑定。選択肢は2つ:
  1. 鑑定資源（霊粉の転生。ラン内消費財、節目商人でも購入可）で
     確定させてから装備する。
  2. 未鑑定のまま装備して賭ける。当たりならそのまま、
     呪いなら外せない＋マイナス効果。解呪は節目商人で高額。
- 深度が上がるほど当たりの質も呪いの重さも上がる。
- スキル/スペル: クラス初期セット＋ラン内の書物ドロップで習得。
  プールはメタアンロックで拡張。

## メタ経済（工房＝アンロックツリー）

- メタ通貨＝素材（既存10種を転用: 獣の牙・硬い皮・毒腺・骨片・霊粉・
  魔石片・鉄片・呪布・黒角・竜鱗）。深度・バイオームで排出種が変わり、
  「あの素材が欲しいからあの層へ」を作る。
- 工房で買えるもの:
  - 新クラス解放
  - 初期装備候補の拡充
  - スキル/スペル・アフィックス抽選プールの拡張
  - 恒久ステータス（上限明示: 各系統+5段まで。深度＝周回数化の防波堤）
  - 利便系（初期鑑定資源+1、帰還の翼初期所持 等）
- **ゴールド廃止**。ラン内買い物も素材払いに一本化。

## 記録・コデックス・契約

- 記録: 自己最深（撤退/死亡別）、クラス別最深、総潜行回数。
- コデックス: 維持。初討伐報酬はメタ素材ボーナスに転生。
- 契約（contracts）: 町の受注制を廃止し、ラン内クエストに転生。
  潜行開始時に1〜2件自動付与（例:「B10到達」「disruptor3体撃破」）、
  達成で素材ボーナス。

## 移行・実装方針

- 旧セーブ: 破棄。移行処理・補償なし。旧セーブ検出時は
  新規開始を案内するだけ。
- モジュール生存マップ:
  - 転用: `src/combat_logic/*`（数式部品）、`src/data/*`
    （敵・装備・アフィックス・スペル）、`src/map_generator.js`、
    `src/seed_rng.js`、`src/styles/*`、resume基盤、コデックス。
  - 転生: 商人（節目商人化）、素材（メタ通貨化）、
    契約（ラン内クエスト化）。
  - 削除: `src/training.js`、roster、remains、蘇生、`src/camp.js`、
    party HUD、party_assemble、町ショップ経済。
- 実装順（各段でプレイアブル維持、issue分割）:
  1. 生成マップのラン毎リシード検証（最大リスクを最初に潰す）
  2. ソロ戦闘コア（行動経済・状態異常・逃走）
  3. 鑑定ギャンブル
  4. メタ工房・素材通貨
  5. 節目フロア・バイオーム・ショートカット
  6. 記録・ラン内クエスト・コデックス転生

## 敵対的評価で確認済みのリスクと対策

- 深度スコア×持ち帰り経済の矛盾 → 撤退＝満額/死亡＝減額で
  押し引き（旧柱2）をローグライト文法へ転生。
- 固定マップ決定との矛盾 → 決定を正式廃棄。生成品質を
  実装順先頭で検証。
- ソロ×状態異常の即死コイントス化 → 状態異常の全面再設計で対処。
- 深度＝周回数化 → 毎ランLv1＋恒久ステ上限＋節目開始の
  素材減額係数で抑止。
- 深層の単調化（無限スケーリングの宿命） → v1はバイオーム再循環で
  割り切り、手応え次第で深層専用コンテンツを追加。
